# go-currency-lab

## What
Exploring Go concurrency fundamentals: goroutines, channels, and deadlocks
This repository documents small experiments to build intuition about flow control and termination behavior

## Why
My goal is to understand concurrency deeply enough to read and modify production-grade Go code (e.g., Docker).

Rather than jusst learning syntax, this repo fucuses on:
- Block behavior
- Termination rules
- Producer-consumer flow
- Deadlock scenarios

## Structure

- /cmd -> runnable experiments
- /experiments -> written observations
